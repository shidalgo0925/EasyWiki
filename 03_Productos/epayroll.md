# EPayRoll

**EPayRoll** es el producto de [[02_Suite/easy_suite|Easy Suite]] para **nómina y gestión de personal**: cálculo de salarios, deducciones, reportes legales y exportación contable.

## Estado en suite

Marca comercial de la suite. **Sin despliegue en CODITO** (jun 2026) — solo documentación wiki y placeholder de proyecto.

La integración con [[03_Productos/en1_platform|EN1]] (empleados como contactos, centros de costo, contabilidad) se define **por contrato**.

## Para qué sirve (visión)

| Área | Descripción |
|------|-------------|
| Nómina periódica | Quincenal/mensual según legislación local |
| Deducciones | Seguro, impuestos, préstamos |
| Reportes | Archivos para banco y ministerio de trabajo (según país) |
| Integración | Envío a contabilidad o ERP del cliente |

## Relación con EN1

- Maestro de **contactos** / empleados puede alimentarse desde EN1.
- Módulo `accounting` en EN1 está reservado para evolución contable avanzada; nómina puede ser producto separado EPayRoll.
