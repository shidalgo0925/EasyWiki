# ePosOne

**ePosOne** es el producto de [[02_Suite/easy_suite|Easy Suite]] orientado al **punto de venta (POS)**: ventas rápidas en mostrador, caja y operación retail, conectado al negocio del cliente.

## Para qué sirve

- Registrar ventas en **mostrador** o tienda física.  
- Trabajar con **productos, precios y clientes** del negocio.  
- Sincronizar o convivir con la plataforma central ([[03_Productos/en1_platform|EN1]]) cuando el proyecto lo requiere (inventario, contactos, reportes).

## A quién va dirigido

- Retail, ferreterías, distribución ligera.  
- Negocios con **caja** y necesidad de ticket / comprobante.  
- Cadenas que ya usan o adoptarán EN1 para back-office.

## Qué obtiene el cliente

| Beneficio | Descripción |
|-----------|-------------|
| Rapidez en caja | Flujo pensado para cola y mostrador. |
| Coherencia de datos | Misma base de productos/clientes que el ERP (si está integrado). |
| Soporte local | Implementación por [[01_Empresa/easytechnology|Easy Technology Services]]. |

## Relación con EN1

| Escenario | Enfoque |
|-----------|---------|
| Solo POS | ePosOne como producto principal. |
| POS + gestión | ePosOne + EN1 (inventario, facturación, membresías). |
