# Integraciones

Easy Suite y [[03_Productos/en1_platform|EN1]] se conectan con sistemas del cliente.

| Tipo | Ejemplo |
|------|---------|
| **ERP Odoo** | [[04_Clientes/modecosa]] — catálogo de seguridad / permisos |
| **Pagos** | Yappy manual, transferencia, PayPal (según proyecto) |
| **Office 365** | Correo y calendario (módulo) |
| **Landings** | Sitios de marketing externos con API pública |
| **App móvil** | Sincronización con APIs EN1 |

Detalle técnico de implementación: `05_Proyectos/` y `06_Arquitectura/`.
